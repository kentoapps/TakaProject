# TakaProject
たかはるAndroidアプリ開発トレーニングプロジェクト

# Gitの使い方
## 課題をはじめる
### プロジェクトをダウンロードする
```
git clone https://github.com/kentoapps/TakaProject.git
```
もしくは
```
git clone git@github.com:kentoapps/TakaProject.git
```

## 課題が終わったら
### 成果を保存する
```
git add .
```
```
git commit -m "タスク1完了"
```

### 成果をアップする
```
git push
```
